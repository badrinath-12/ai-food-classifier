# 🍔 AI Food Classification Web Application

## 📌 Project Overview
This is a Flask-based web application that uses a pretrained deep learning model from Hugging Face to classify food images. Users can upload an image and the system predicts the food type along with confidence scores.

---

## 🚀 Features
- 📷 Image upload functionality
- 🤖 AI-based food classification using pretrained model
- 📊 Displays top predictions with confidence scores
- 🖥️ Simple and responsive web UI
- ⚡ Fast inference using Hugging Face pipeline

---

## 🧠 Tech Stack
- Python
- Flask
- Hugging Face Transformers
- PIL (Python Imaging Library)
- HTML, CSS, JavaScript
